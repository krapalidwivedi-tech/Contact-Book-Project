# Contact Book Project

A simple Contact Book application built using Python and SQLite3.

## Features

* Add new contacts
* View all contacts
* Search contacts
* Update contact details
* Delete contacts
* Store data using SQLite3 database

## Technologies Used

* Python
* SQLite3

## Project Structure

* `code.py` - Main application code
* `contact_book.db` - SQLite database file

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

```bash
python code.py
```
